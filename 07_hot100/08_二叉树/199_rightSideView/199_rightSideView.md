# [199_二叉树的右视图](https://leetcode.cn/problems/binary-tree-right-side-view/)

难度：中等

## 问题描述：

给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

 

**示例 1：**

```java
输入：root = [1,2,3,null,5,null,4]
输出：[1,3,4]
```
