# [94_二叉树的中序遍历](https://leetcode.cn/problems/binary-tree-inorder-traversal/)

难度：简单

## 问题描述：

给定一个二叉树的根节点 `root` ，返回 *它的 **中序** 遍历* 。

**示例 1：**

![img](../../assets/imgs/inorder_1.jpg)

```java
输入：root = [1,null,2,3]
输出：[1,3,2]
```

**示例 2：**

```java
输入：root = []
输出：[]
```

**示例 3：**

```java
输入：root = [1]
输出：[1]
```
