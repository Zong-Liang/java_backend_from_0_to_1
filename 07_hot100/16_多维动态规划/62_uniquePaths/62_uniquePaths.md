# [62_不同路径](https://leetcode.cn/problems/unique-paths/)

难度：中等

## 问题描述：

一个机器人位于一个 `m x n` 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

**示例 1：**

![img](../../assets/imgs/1697422740-adxmsI-image.png)

```java
输入：m = 3, n = 7
输出：28
```

**示例 2：**

```java
输入：m = 3, n = 2
输出：3
解释：
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -> 向下 -> 向下
2. 向下 -> 向下 -> 向右
3. 向下 -> 向右 -> 向下
```

**示例 3：**

```java
输入：m = 7, n = 3
输出：28
```

**示例 4：**

```java
输入：m = 3, n = 3
输出：6
```
