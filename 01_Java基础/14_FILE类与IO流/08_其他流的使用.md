# 08_其他流的使用

标准输入输出流：

- `System.in`：标准输入流，默认从键盘输入。
- `System.out`：标准输出流，默认从显示器输出 (控制台)。

可以调用如下方法，修改输入流和输出流的位置：

- `setIn(InputStream is)`
- `setOut(PrintStream ps)`

打印流：

- `PrintStream`
